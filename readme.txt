Le projet contient 4 pages de struture different :

1-La page "index.html" : c'est la page d'acceuil ,  elle contient des liens vers toutes les autres pages
2-Dossier quiz : Contient la page "quiz.html" 
3-Dossier Forms : contient les deux formulaires "inscription" et "connexion"
4-Dossier cours : Contient 3 dossiers avec une page de la meme structure (les cours differents)
       Dossier HTML : 3 pages ( une page pour chaque chapitre ) 
       Dossier CSS : 3 pages ( une page pour chaque chapitre ) 
       Dossier JS : 3 pages ( une page pour chaque chapitre )

Concernant le CSS : 
-Le document "Style.css" est le fichier de style general de la page index.html 
-Le document "quiz.css" fichier de style de quiz
-Le document "courstyle.css" fichier de style des cours
-Le document "formstyle.css" fichier de style des forms
TOUT LES AUTRES PAGES UTILISENT "STYLE.CSS" pour avoir le meme header et footer

Concernat Javascript :
-il y'a un seul fichier javascript dans le dossier "Quiz" qui contient la function note() qui calcule la note
et la afficher.
-il y'a aussi un script JS dans la page "index.html" pour le diaporama.
